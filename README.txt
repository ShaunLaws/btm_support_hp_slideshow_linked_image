
-- SUMMARY --

The BTM Support HP Slideshow Linked Image module is the support module to the
BTM Features HP Slideshow Linked Image Features-generated module that eases the
deployment of a home page slideshow with linked images.

The BTM Support HP Slideshow Linked Image module simply makes sure that the
Jquery Cycle Plugin module, that is used by the BTM Features HP Slideshow Linked
Image module to generate the slideshow, and the custom javascript file that
calls the Cycle function are included on the home page.


-- REQUIREMENTS --

None.


-- INSTALLATION --

* This module is dependent on the following modules:

  - Jquery Plugin - contains the cycle plugin that is used by this module

* Download this dependent module to your site if it is not already installed

* Install as usual, see http://drupal.org/node/70151 for further information.
